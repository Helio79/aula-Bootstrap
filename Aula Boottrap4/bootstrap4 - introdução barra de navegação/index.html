Base nav

A navegação disponível no Bootstrap compartilha marcações e estilos gerais, da .navclasse base aos estados ativo e desabilitado. Troque as classes do modificador para alternar entre cada estilo.O .navcomponente básico é construído com flexbox e fornece uma base sólida para a construção de todos os tipos de componentes de navegação. Inclui algumas substituições de estilo (para trabalhar com listas), algum preenchimento de link para áreas de acertos maiores e estilo básico desativado.
